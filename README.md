# bone-dyali

A small static website built with Vite and Tailwind CSS.

Quick start

1. Install dependencies

   npm install

2. Start the development server

   npm run dev

3. Build for production

   npm run build

4. Preview the production build

   npm run preview

Project structure

- index.html — entry point
- src/ — source files
- dist/ — production build output

Notes

- Configured with Vite, Tailwind CSS, and PostCSS (see vite.config.js, tailwind.config.js, postcss.config.js).
